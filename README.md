# Smart Market

Smart Market is a local online marketing platform built to help Ghanaian businesses showcase their products and services while enabling users to easily discover nearby vendors.

The platform focuses on simplicity, accessibility, and real-world usability for small businesses and local markets.

## Features
- Business listings with detailed profiles
- Category-based and location-aware discovery
- User authentication
- Real-time data updates
- Responsive, mobile-first design
- Scalable architecture for future growth

## Tech Stack
- React.js
- Tailwind CSS
- Supabase (Authentication & Database)
- JavaScript
- Git & GitHub
